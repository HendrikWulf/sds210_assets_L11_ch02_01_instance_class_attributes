Interactive Explorer: Class vs. Instance Attributes

A Class Attribute (like CRS) is shared globally by all objects. Changing it affects every instance simultaneously. An Instance Attribute (like latitude) is unique to a specific object. Modifying it only affects that single encapsulated entity.
